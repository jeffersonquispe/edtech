🔴 CRÍTICO - Requerido para continuar:

1. Node.js 16.20.0 → Necesita 20.9.0+
   Solución: Ver REQUIREMENTS.md (3 opciones para actualizar)
   Impacto: No puede compilar Next.js ni instalar shadcn/ui

2. Migraciones no aplicadas en Supabase
   Solución: Ver SETUP_SUPABASE.md (copiar SQL a SQL Editor)
   Impacto: BD vacía, endpoints retornarán error

🟡 MODERADO - Necesario para Fase 3:

3. Tipos TypeScript sin generar (database.ts)
   Solución: supabase gen types typescript
   Impacto: Sin autocompletar en queries

4. shadcn/ui no instalado
   Solución: npx shadcn@latest init
   Impacto: Necesario para componentes UI

                        PRÓXIMOS PASOS
================================================================================

1️⃣ ACTUALIZAR NODE (crítico)
   Leer: REQUIREMENTS.md
   Comando: nvm use 20 (o descargar v20 LTS)
   Verificar: node --version

2️⃣ APLICAR MIGRACIONES EN SUPABASE (crítico)
   Leer: SETUP_SUPABASE.md
   Ir a: https://app.supabase.com > SQL Editor
   Acción: Copiar TODO el SQL y ejecutar

3️⃣ GENERAR TIPOS TYPESCRIPT
   Comando: cd backend && supabase gen types typescript > types/database.ts
   Luego: cp backend/types/database.ts frontend/types/database.ts

4️⃣ INSTALAR SHADCN/UI
   Comando: cd frontend && npx shadcn@latest init
   Responder: "default" para todas las opciones

5️⃣ INICIAR FASE 3 (Frontend)
   Crear páginas:
   - /login (Supabase Auth)

   - /dashboard/student (mis cursos)
   - /learn/[courseId] (lecciones)
   - /dashboard/instructor (gestión)
